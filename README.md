# 简介

# **此后仅在周日晚上更新，更新日志在README最底下**

如想测试mod可以打开target文件夹下载jar文件并丢到杀戮尖塔游戏根目录的mods文件夹中

这是一个正在开发的杀戮尖塔mod项目，人物为知名豹白兼某校塔群群主二硫键，目前仅由我一人开发

预计最终会包含几百张卡与上百个遗物，以及数位可玩人物，还有对其他mod的兼容与适配（比如联机mod）

V1.0.0版本预计2026年暑假上传

# mod介绍

目前仅有1位人物：二硫键

**机制1：骰子**

一种类似充能球的物体

- 骰子槽位固定为5

- 骰子会在回合结束时全部激发

- 骰子效果受其点数影响，出于平衡性考虑，负面效果相对较小

- 骰子造成伤害类型比较特殊，详见游戏内

**机制2：罪孽与美德**

已重构

打出部分卡牌会获得罪孽与美德，罪孽与美德相互抵消（就像集中和负集中一样）

二硫键头顶上的能量条为罪孽值条，红色代表此时为罪孽，蓝色代表此时为美德

战斗结束时根据你的罪孽或美德值：

- 每10层罪孽使你失去1点最大生命值

- 每10层美德使你获得1点最大生命值

特别的，当你在战斗中罪孽层数达到50或美德层数达到50时会锁定罪孽/美德层数不再改变，且会在战斗结束时额外赠与你恶魔赠礼或神圣赠礼：

- 恶魔赠礼：拾起时失去5点最大生命值，选择并移除一张牌

- 神圣赠礼：拾起时获得5点最大生命值，选择祝福一张牌

祝福为新增机制，定位是增强白卡和其他“牢九门”之类的强度尴尬的卡，未来会完善相关机制

目前二硫键获得罪孽的方法远大于获得美德的方法，未来可能会完善美德获得相关途径

**机制3：卡包**

二硫键本体只有45张牌，剩下30张由每局游戏开始时选择的卡包凑齐一局游戏的卡池：

- 卡包均为根据群友的网络形象创作，各自都有自己的机制和侧重点

- 每个群友卡包有3个变种，分别侧重数值，运转与上限

- 群友卡包中有5张会在所有变种中出现，每个卡包还各自有5张卡，用于调整卡池结构侧重点

- 游戏开始时会展示3个来自不同群友卡包，从左到右分别为侧重数值、运转、上限的卡包变种，玩家需选择1个卡包；重复3次

**机制4：身份**

二硫键拥有两种身份：群主身份与恶魔之焰身份

恶魔之焰身份会加强七宗罪相关卡，群主身份会加强群友卡包的卡
