# 简介

如想测试mod可以打开target文件夹下载jar文件并丢到杀戮尖塔游戏根目录的mods文件夹中

这是一个正在开发的杀戮尖塔mod项目，人物为知名豹白兼某校塔群群主二硫键，目前仅由我一人开发

预计最终会包含几百张卡与上百个遗物，以及数位可玩人物，还有对其他mod的兼容与适配（比如联机mod）

V1.0.0版本预计2026年暑假上传

# mod介绍

目前仅有1位人物：二硫键

**机制1：骰子**

一种类似充能球的物体

- 骰子槽位固定为5

- 骰子会在回合结束时全部激发

- 骰子效果受其点数影响，出于平衡性考虑，负面效果相对较小

- 骰子造成伤害类型比较特殊，详见游戏内

**机制2：罪孽与美德**

这部分考虑重构，目前版本为：

- 二硫键部分牌为七宗罪相关卡，打出这些卡会获得罪行

- 每当拥有不少于10层罪行时，失去所有罪行并获得1层罪孽

- 每次获得罪孽时，失去等量的最大生命值，并在回合结束时受到等同于罪孽层数的伤害

- 一些衍生卡为美德卡，每当获得美德时获得等量的最大生命值

**机制3：卡包**

二硫键本体只有45张牌，剩下30张由每局游戏开始时选择的卡包凑齐一局游戏的卡池：

- 卡包均为根据群友的网络形象创作，各自都有自己的机制和侧重点

- 每个群友卡包有3个变种，分别侧重数值，运转与上限

- 群友卡包中有5张会在所有变种中出现，每个卡包还各自有5张卡，用于调整卡池结构侧重点

- 游戏开始时会展示3个来自不同群友卡包，从左到右分别为侧重数值、运转、上限的卡包变种，玩家需选择1个卡包；重复3次

**机制4：身份**

二硫键拥有两种身份：群主身份与恶魔之焰身份

恶魔之焰身份会加强七宗罪相关卡，群主身份会加强群友卡包的卡
